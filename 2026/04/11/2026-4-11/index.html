  1. 什么是 Flex？
    一句话：Flex = 一维布局（要么排一行，要么排一列）
    它能干什么？
    把元素排成 一行（水平排列）
    把元素排成 一列（垂直排列）
    让元素 居中、平分、自动撑开
    对比写法（超好记）
    css
    /* 水平排列 */
    display: flex;

/* 垂直排列 /
display: flex;
flex-direction: column;
你会用到的场景
导航栏水平排列
按钮组水平对齐
Hero 区文字居中
让一个盒子上下左右居中
你的代码里出现过
css
display: flex;
align-items: center; / 上下居中 /
justify-content: center; / 左右居中 /
2. 什么是 Grid？
一句话：Grid = 二维布局（可以同时控制行和列，像 Excel 表格）
它能干什么？
同时控制 多行 + 多列
实现卡片网格、瀑布流、复杂页面布局
对比写法
css
display: grid;
grid-template-columns: 1fr 1fr 1fr; / 三列等宽 /
grid-gap: 20px; / 格子之间间距 */
看起来像
plaintext
┌─────┬─────┬─────┐
│ │ │ │
├─────┼─────┼─────┤
│ │ │ │
└─────┴─────┴─────┘
你代码里的问题（关键）
你之前写的：
css
display: grid;
grid-template-columns: minmax(0, 3fr) minmax(0, 2fr);
这就是 双列布局！
意思是：
左边占 3 份空间
右边占 2 份空间
内容只能在左列，无法全局居中
所以你才看到 左右错位、文字只在左半部分。
3. 什么是双列布局？
一句话：双列布局 = 一个容器里有左右两列
两种常见写法
Grid 双列
css
display: grid;
grid-template-columns: 1fr 1fr;
Flex 双列
css
display: flex;
子元素分别占 50%
视觉对比
plaintext
[ 左列 ] [ 右列 ]
为什么你报错？
你把 Hero 区做成了 双列布局，但你想要 全局居中，这本身就矛盾。
