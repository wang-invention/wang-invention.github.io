✅ v-if=”isTrue”

  • 核心：条件渲染，布尔值为 false 时「直接不生成 DOM 元素」
  • 特点：按需渲染，切换开销大（销毁 / 重建 DOM），适合不频繁切换的场景

✅ v-show=”isTrue”

  • 核心：条件显示，布尔值为 false 时「通过 CSS (display: none) 隐藏元素」
  • 特点：DOM 始终存在，切换开销小，适合频繁切换的场景

✅ v-on:xxx → 简写 @xxx

  • 核心：事件绑定指令，监听 DOM 原生事件 / 组件自定义事件
  • 常用场景：@click="submit"（点击）、@keyup.enter="submit"（回车）、@input="handleInput"（输入）
  • 配套修饰符：@click.stop（阻止冒泡）、@click.prevent（阻止默认行为）、@submit.prevent（表单阻止刷新）

✅ v-bind:xxx → 简写 :xxx

  • 核心：数据单向绑定，仅「Model → View」（数据变，视图同步变；视图改，数据不变）
  • 常用场景：:src="imgUrl"、:href="link"、:class="clsObj"、:style="styleObj"、:title="msg"

Vue的生命周期

beforeCreate–>Create–>BeforeMount—>mount->beforeUpdate—>update—->destory

create方法已加载，html模板已加载，数据已加载，但是数据为渲染，mount渲染数据，beforeUpdate数据更新但是模板为更新，update 模板更新

. 普通插槽

  • 父组件 → 子组件 传内容
  • 父组件把 HTML / 文本 放到子组件的 <slot> 里
  • 父组件不能用子组件的数据

一句话：

父填内容，子展示，父拿不到子的数据。

作用域插槽（slot-scope / #default）

  • 子组件 → 父组件 传数据
  • 子组件在 slot 上绑定属性：<slot :data="data" :node="node"/>
  • 父组件用 #default="scope" 接收
  • 父组件可以使用子组件内部的数据

一句话：

子把数据传给父，父自定义渲染。
